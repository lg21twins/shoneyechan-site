# Haru Anbu Showcase v8 Bundle

## 폴더 구조

- `07_디자인/`
  - 디자인 산출물 및 mockup 작업 폴더
  - `mockup/` 안에 `v1_의료진웹`, `v11_보호자앱`, `v11_요양보호사앱`, `v12_환자앱`, `v6_전체화면` 등의 화면별 구성
- `Mockup-templates/`
  - 기기/템플릿 이미지 및 mockup 템플릿 자료
- `docs/`
  - 리뷰 문서와 발표 스크립트
- `발표/`
  - 발표 관련 문서, 오디오, 큐시트
- `_archive/`
  - 백업본과 비교용 파일 보관
  - `_archive/README.md`로 보관 기준 명시
- `07_디자인/mockup/`
  - 화면별 mockup 작업물
  - `07_디자인/mockup/README.md`로 서브 구조 안내
- `docs/_drafts/`
  - 발표 초안 보관
  - `docs/_drafts/README.md`로 초안 관리 안내
- `haru-anbu-showcase-v8.html`
  - 대표 showcase HTML 파일

## 정리 포인트

1. 작업 파일은 `07_디자인/` 아래에 유지
2. 템플릿은 `Mockup-templates/`에 따로 보관
3. 발표 자료는 `발표/`로 분리
4. 보관/백업 파일은 `_archive/`에만 보관
5. 각 폴더별 `README.md`를 추가해 사용성과 이해도를 높임
