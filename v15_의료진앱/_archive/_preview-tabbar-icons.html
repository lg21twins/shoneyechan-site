의료진앱 탭바 · 아이콘 통일
회진 + 인계(SBAR) 후보 각 5종
현재 stethoscope(회진)와 arrow-swap(인계)는 라인 톤이 강해 home/chat bubble의 충만한 실루엣과 충돌. 다른 두 탭(지금·소통)과 시각적 무게가 맞는 fill 형태로 5개씩 후보 시각화.
지금 (home-24-filled)
소통 (chat-bubble)
교체 대상회진 · 인계
A · 회진 아이콘 후보 5종
현재: fluent:stethoscope-24-filled — 청진기 튜브가 본질적으로 선이라 실루엣 부피가 작음. 후보는 침대·차트·심박 등 더 묵직한 매스를 가진 fill 아이콘.
A1 · 침상 라운드 fluent:bed-24-filled
침상 = 환자 회진의 가장 직관적 메타포. 매스 강함
A2 · 차트 + 펄스 fluent:clipboard-pulse-24-filled
회진 차트 + 바이탈 펄스. 의료 컨텍스트 명확
A3 · 환자 카드 fluent:notepad-person-24-filled
환자 노트 + 사람 실루엣. 환자 중심성 강조
A4 · 회진 체크 fluent:clipboard-task-list-ltr-24-filled
회진 체크리스트 메타포. 실행적 느낌 강함
A5 · 하트 펄스 fluent:heart-pulse-24-filled
하트 매스 + 펄스. 의료 + 케어 톤. 단점: 응급 색채
B · 인계(SBAR) 아이콘 후보 5종
현재: fluent:arrow-swap-24-filled — 화살표 두 개만 있어 실루엣이 가볍고 라인적. 후보는 문서·노트 + 화살표/사람 조합으로 매스 확보.
B1 · 사람 교대 fluent:people-swap-24-filled
교대조 핸드오프 직관. 사람 매스 충분
B2 · 문서 인계 fluent:document-arrow-right-24-filled
SBAR 문서 전달. 문서 매스 + 방향성 명확
B3 · 클립보드 인계 fluent:clipboard-arrow-right-24-filled
회진 카드와 시리즈감(클립보드 계열). 단점: 회진 후보 A2/A4와 형제
B4 · 노트 편집 fluent:notepad-edit-24-filled
SBAR 초안 작성 메타포. 노트 매스 강함
B5 · 패스 / 공유 fluent:share-24-filled
가장 가볍고 추상적. 노드+화살표 실루엣
C · 추천 조합 (회진 + 인계 동시 적용)
시리즈 충돌(둘 다 클립보드 등) 피하면서 매스가 균형 잡힌 조합 3가지.
추천 1 · 침상 + 사람 교대 bed + people-swap
"환자 침상 라운드 → 교대조에 인계" 스토리 가장 직관적
추천 2 · 차트 + 문서 전송 clipboard-pulse + document-arrow-right
"바이탈 차트 → SBAR 문서 전송" — 의료 문서 워크플로 강조
추천 3 · 환자 노트 + 노트 편집 notepad-person + notepad-edit
같은 노트 패밀리로 시리즈감. 단점: 두 아이콘이 다소 비슷
