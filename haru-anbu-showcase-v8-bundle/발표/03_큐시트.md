## v8 사이트 흐름 메모 (큐 보기 전 한 번)

**v8 섹션 순서 (위→아래)**: Hero → Overview → Problem → App-preview → Direction → Design → **#flow-overlay (SOS)** → Service-features → Feature-bridge-visual → Flagship-features → Features → **#features-by-role (5앱)** → Epilogue

**시그니처 4모먼트만 멈춤**: SOS(`#flow-overlay`) · 5앱(`#features-by-role`) · Demo · Epilogue

**스크롤로 빠르게 통과**: Overview · Problem · App-preview · Direction · Design · Service-features · Feature-bridge · Flagship-features

**SOS 앵커 점프**: 인트로 직후 URL을 `#flow-overlay`로 점프해서 SOS를 시그니처 오프닝 자리에 놓음. (Hero+Meet 멘트 자리는 큐시트 본문 참조)

> ⚠️ **검증 필요**: v8은 와이어프리뷰와 섹션 순서·ID가 다름. 실제 사이트를 띄워 놓고 한 번 큐를 따라가며 (1) 앵커 점프 동작, (2) 스크롤 통과 시 어색하지 않은지, (3) Hero+Meet 멘트 자리 적정성을 검증할 것.

## 비상 대처

| 상황 | 대처 |
|------|------|
| Orb mp3 안 들림 | Shon이 직접 같은 내용 읽기 ("AI가 이렇게 말합니다 — ...") |
| 화면 멈춤 / iframe 안 뜸 | "잠시 직접 화면 캡처로 보여드리겠습니다" 하고 백업 PNG로 |
| 시간 부족 | Persona·Design Language 외에 추가로 Impact 줄임 |
| 시간 남음 | Demo에서 한 탭 더 시연 (간호사 또는 요양보호사) |
