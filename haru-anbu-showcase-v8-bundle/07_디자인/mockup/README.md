# mockup

`07_디자인/mockup/`는 화면별 mockup HTML, CSS, 이미지, 문서를 담고 있는 폴더입니다.

하위 폴더:
- `v1_의료진웹/` : 의료진 웹 화면 mockup
- `v11_보호자앱/` : 보호자 앱 화면 mockup
- `v11_요양보호사앱/` : 요양보호사 앱 화면 mockup
- `v12_환자앱/` : 환자 앱 화면 mockup
- `v6_전체화면/` : 전체 화면 mockup

정리 팁:
- 화면별 버전이 섞이지 않도록 폴더 단위로 유지
- 완성본 HTML은 `ready` 또는 `final` 폴더로 별도 분리 가능
