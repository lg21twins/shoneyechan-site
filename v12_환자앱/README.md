# 하루안부 · 환자 앱 v12

**상태**: WIP — POC 1페이지 시작 (p01-today.html)
**대상 기기**: 태블릿 (iPad 10", Galaxy Tab A) — 거치형 사용

---

## 플랫폼 확정

- **모바일 PWA** (태블릿 폼팩터 우선)
- `<html data-role="patient" data-platform="mobile">`
- 태블릿 거치형을 기본으로 디자인하되, 스마트폰에서도 동작하게 반응형

환자는 `data-role="patient"` 테마에서 자동으로:
- 본문 18px (기본 16px 대비 +2px)
- 헤드라인 20px
- 터치 타깃 56px (기본 44px 대비 크게)
- 액션 행 64px

추가로 태블릿 거치를 위해 카드·여백을 키우고, 핵심 정보는 한눈에 보이게 카드 한 개 = 한 화면 정도로 **한 번에 1가지 정보**만 강조.

## 설계 원칙

1. **큰 글자·큰 버튼**: data-role="patient" 자동 오버라이드 활용
2. **말 수 적게**: 한 화면에 메시지 1개, 액션 1개
3. **조작 오류 내성**: 취소가 무조건 있음, 실수 방지 확인 1회
4. **가족 얼굴 보이기**: 가족 아바타 큼지막하게
