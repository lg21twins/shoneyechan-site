마이크 자리에 넣을 음파 아이콘 후보 6종

실제 크기(입력바 자리) + 아이콘 확대 + 이름 + 출처 코드

A
Sound Wave Circle (Filled)
원형 안에 웨이브. 둥글고 부드러움. 지금 ai-input-dot과 톤 유사
fluent:sound-wave-circle-24-filled
B
Waves (물결)
부드러운 사인 웨이브 3줄. 차분하고 감성적
material-symbols:waves
C
Graphic EQ (Rounded)
이퀄라이저 바 3개. 클래식한 오디오 시그널
material-symbols:graphic-eq-rounded
D
Lucide Audio Waveform
사이드 감소형 웨이브. Siri스러운 대칭
lucide:audio-waveform
E
Voiceprint (Siri 스타일)
굵은 바들이 불규칙 배열. Siri·ChatGPT 음성모드 톤
ri:voiceprint-fill
F
Mingcute Voice Line
마이크 + 웨이브 콤비. 음성입력 의미 가장 직관적
mingcute:voice-line

펄스 애니메이션은 샘플용. 실제 적용 시 입력시에만 움직이게 할 수 있음.
