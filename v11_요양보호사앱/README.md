# 하루안부 · 요양보호사 앱 v11

**상태**: 1차 목업 정비 완료 (4탭 + 통합 SOS · 2026-05-10)
**디자인 기준**: `07_디자인/tokens/tokens.css` + `system/app.css` + `system/components.css` (디자인 시스템 v3.1).
보호자앱 v9.5의 시각 언어를 일부 공유하지만, 카드/탭바/헤더/SOS는 본 앱 전용 `caregiver.css`와 디자인 시스템 컴포넌트로 통일한다.

---

## 플랫폼

- 모바일 PWA 전용 (데스크톱 웹 미지원)
- `<html data-role="caregiver" data-platform="mobile">`
- 셸: `07_디자인/tokens/template.html` 패턴 + 본 폴더 `caregiver.css`/`caregiver.js`
- 케어 그린 accent는 토큰의 `--color-accent`(green-500 `#22C55E`) 기반. 화면 코드에서 hex 직접 사용 금지

요양보호사는 의료진 직군(의사·간호사·요양보호사)과 accent 컬러를 공유하지만, **현장(환자 옆)에서 손에 들고 쓰는 도구**라 모바일 전용이다. 의료진 웹 대시보드의 사이드바+상단바 레이아웃은 사용하지 않는다.

---

## 디자인 규칙 (필수)

- UI와 문서 전체에서 **이모지 금지** (소셜·정서 표현 포함). 아이콘이 필요하면 `iconify-icon`의 `fluent:*` 계열만 사용.
- 색·간격·라운드·그림자는 모두 `tokens.css` 시멘틱 토큰을 통해 사용. hex/rgba 직접 사용 금지.
- 모든 인터랙티브 컨트롤의 hit area는 **48px 이상**. visual icon이 작더라도 button padding으로 영역을 확보.
- 카드는 시스템 4종(`card`, `card-action`, `card-alert`, `card-hero`) 중심. glass(`card-glass`)는 탭바·모달·강조 한정.
- 폰트는 디자인 시스템 chain 그대로 사용 — 한국어는 Pretendard Variable, 중국어 폴백은 Noto Sans SC. 본 앱은 다국어 사용자(중국 출신 보호사) 비율을 고려해 Noto Sans SC를 추가 로드한다.

## 화면 인벤토리 (4탭 · IA v1.0 기준)

| ID | 파일 | 탭 | 주요 구성 |
|---|---|---|---|
| C-01 | `c01-today.html` | 오늘 | 인사말·근무 진행률·관찰 카드·지금 할 일·AI 인수인계·담당 4명 상태 |
| C-02 | `c02-checklist.html` | 기록 | 환자 칩·시간대 세그먼트·8카테고리 그리드·**8단계 케어 기록 스텝퍼**·빠른 기록 시트 |
| C-03 | `c03-sotong.html` | 소통 | 인수인계 미확인 detail·이상감지 detail·메시지·보호자 공유 preview |
| C-04 | `c04-mypage.html` | 마이 | 프로필·오늘 근무·주간 일정·서비스 기록·자격·알림 스위치 |
| C-04S | `c04-schedule.html` | (push) | 주간 일정 detail (탭바 없음) |
| C-SOS | 전역 FAB | 모든 탭 | 탭바 우측 빨간 원형. 탭=유형 시트, 3초 길게 누름=즉시 발신 (countdown·취소·로그) |

탭 라벨(`오늘`·`기록`·`소통`·`마이`)은 모든 화면에서 동일 위치·동일 텍스트로 노출한다.

## 통합 SOS

- 모든 탭에서 같은 컴포넌트(`caregiver.js`의 `HaruSOS`)를 사용한다.
- 짧게 누름 → 유형 선택 sheet (낙상·의식 저하·호흡 곤란·기타 음성)
- 3초 길게 누름 → 1초 confirmation countdown 후 발신. 발신 직전 `취소` 제공
- 발신 후에는 발송 대상(간호사·보호자·센터)과 시각이 toast + 소통 탭 알림 로그에 남는다

## 8단계 스텝퍼 (c02)

한 화면 한 항목, 한 손 조작 기준. 각 단계는 `.st-step[data-step]`로 스위칭한다.

1. 바이탈 — 혈압·체온·맥박·혈당 (keypad sheet)
2. 식사 — 식사량 슬라이더·형태 세그먼트·수분
3. 투약 — 약 체크 그리드·거부 메모
4. 위생 — 구강·세수·세발·목욕·손발 체크 그리드
5. 활동 — 보행·체위·프로그램 체크·총 활동시간
6. 배변 — 대변·소변·기저귀 (민감 · 별도 공유 설정)
7. 인지·정서 — 5단계 텍스트 칩(`우울`·`무기력`·`양호`·`활기`·`불안`)·대화 요약·음성
8. 특이사항 — 메모·사진·간호사 즉시 알림 토글

저장 후에는 환자 row의 진행률, 완료 count, 동기화 상태(`서버 저장`·`오프라인 대기`)가 즉시 갱신된다.
